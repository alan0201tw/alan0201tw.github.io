Game Project Overview

2018-05-08
GameDev/Design

I’ve been developing games since my first year in NCTU. I started by participated in the game design club in out university, NCTUGDC for short. I learned a lot about game developing in Unity and C# scripting at that time, but I didn’t really advance my skill until I really start making games.

So I’m going to fastly walk through some of my game projects in this article.

All these projects are implemented in Unity engine and C#.

Gobang

My first game project is a Gobang game with a very simple AI. I use colliders to detect click event and place a piece accordingly.

The AI I made for this game is quite simple, therefore is quite useless (LOL). I implemented a very simple AI with greedy algorithm approach.

Gravity

A shooting defense game. Player controls a battleplane and try to protect the planet by shooting all the fireballs coming towards the planet.

Destination (UI part)

My first game project as a team member. I mainly do the UI part of the game, but I also helped coding some behavior scripts for cameras and audios.
